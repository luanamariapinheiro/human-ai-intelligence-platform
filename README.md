# 🚀 Human + AI Intelligence Platform

## 📌 Overview

Human + AI é um projeto de Data Science e Machine Learning desenvolvido para analisar como a Inteligência Artificial pode potencializar diferentes profissões com base em habilidades cognitivas e competências profissionais.

O projeto utiliza dados públicos do O*NET Database, técnicas de clustering, PCA (Principal Component Analysis) e visual analytics para identificar padrões profissionais associados à amplificação através da IA.

---

# 📊 Dataset

Fonte oficial:

* O*NET Database
* https://www.onetcenter.org/database.html

Dados utilizados:

* Abilities
* Skills
* Cognitive Attributes

---

# 🧠 Machine Learning

O projeto utiliza:

## ✅ KMeans Clustering

Para segmentar profissões em grupos cognitivos semelhantes.

## ✅ PCA (Principal Component Analysis)

Para redução dimensional e criação do mapa cognitivo profissional.

## ✅ AI Enhancement Score

Score autoral baseado em:

* Critical Thinking
* Active Learning
* Deductive Reasoning
* Oral Expression

# 📊 Insights

Os resultados indicam que profissões com maior intensidade cognitiva e capacidade analítica apresentam maior potencial de amplificação através da Inteligência Artificial.

Entre as profissões com maiores scores:

* Physicists
* Mathematicians
* Judges
* Biochemists
* IT Project Managers
